# 编译指南

由于系统环境中未检测到可用的命令行 Gradle 构建工具，建议您直接使用 Android Studio 打开项目进行编译。

## 步骤

1.  打开 **Android Studio**。
2.  选择 **Open** (打开项目)。
3.  导航到: `C:\Users\Wing\.gemini\antigravity\scratch\FolderPlayer`.
4.  点击 **OK**。
5.  Android Studio 会自动检测并下载所需的 Gradle Wrapper 和依赖项。
6.  等待 Sync 完成后，点击顶部的绿色 **Run** 按钮 (三角形图标) 即可在模拟器或真机上运行。

## 注意事项
- 您的 Android SDK 路径已自动配置在 `local.properties` 中: `C:\Users\Wing\AppData\Local\Android\Sdk`。
- 项目最低支持 Android 8.0 (API 26)。
